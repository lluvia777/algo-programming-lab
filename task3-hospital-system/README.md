# Task 3: Hospital Appointment and Test Result System

## System Description
This project integrates two distinct healthcare modules into a single application accessible via a Main Menu:
1. **Appointment System:** Verifies patient identity, allows clinic and doctor selection, displays available time slots, and confirms the appointment by simulating an SMS notification.
2. **Test Result System:** Verifies identity, checks for the existence and readiness of test results, displays them, and offers a PDF download option.

## Personal Notes
In this task, I learned how to modularize complex algorithms by using a Main Menu to route users to different subsystems. Graphically, I discovered how to use `subgraph` structures in both Graphviz (DOT) and Mermaid to visually group related processes (clusters). This technique is incredibly useful for keeping large, multi-module diagrams organized and highly readable.
